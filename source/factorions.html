Задача

Напишите на JavaScript функцию, которая выводит список всех чисел, которые равны сумме факториалов своих цифр. Пример такого числа:
4! + 0! + 5! + 8! + 5! = 40585

Искомые числа попадают под определение факториона. В десятичной системе их всего четыре штуки. Более подробную информацию см на Факторионы

Основываясь на описаных в статье фактах, можно вычислить их, например, таким кодом. 

     function sumMembersFactorials(num) {
        var digits = ("" + num).split("");
        var sum = 0;

        for (var i = 0; i < digits.length; i++) {
          sum += getFactorial(digits[i]);
        }

        return sum;
      }

      function listOfFactorions() {
        var max_number = 7 * getFactorial(9);

        for (var i = 1; i <= max_number; i++) {
          if (i == sumMembersFactorials(i)) {
            console.log(i)
          };
        }
      }

      function getFactorial(num) {
        if (num <= 1) {
          return 1;
        } else {
          return num * getFactorial(num - 1);
        }
      }

      listOfFactorions();
